# 📝 Mini WordPad (Java Swing)

A simple and powerful **desktop text editor** built using Java Swing.
This project demonstrates GUI development, file handling, and text formatting in Java.

---

## 🚀 Features

* 📂 File Operations (New, Open, Save, Save As)
* ✂️ Cut, Copy, Paste
* 🔍 Find & Replace
* 🎨 Text Formatting (Bold, Italic, Underline)
* 🎯 Font Family & Size Selection
* 🌈 Text & Background Color Change
* 📊 Word Count
* 🕒 Insert Date & Time
* 🖥️ Clean and User-Friendly GUI

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/amritt2006/Mini-WordPad-Java.git
```

2. Compile and run:

```bash
javac MiniWordPad.java
java MiniWordPad
```
